# Data Notes

This folder contains structured summary data prepared from the published production scheduling optimization paper.

## Included files

### `benchmark_instances.csv`
Thirteen benchmark parallel-machine scheduling instances from the paper.

### `ga_results_summary.csv`
Reported GA results, including average total tardiness, standard deviation, number of times the optimal value was found, average time to best solution, average generations, and percentage deviation from optimal.

### `mathematical_model_vs_ga_time.csv`
Comparison of the mathematical model and the proposed GA in terms of total tardiness and solution time.

### `ga_strategy_hypothesis_tests.csv`
Summary of the statistical tests used to evaluate the immigration operator, surrogate fitness function, and expanded initial population.

### `mutation_strategies.csv`
Description of the four mutation strategies used in the crossover-free GA.

### `project_key_metrics.csv`
Portfolio-ready summary metrics.

## Note

These files are prepared as project summary/demo files from values reported in the paper and are intended for educational, research, and portfolio demonstration purposes.
